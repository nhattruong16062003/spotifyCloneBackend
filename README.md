# Spotify Clone Backend

This is the backend for a Spotify clone application built with Django and Django REST Framework. The backend provides APIs for managing songs, artists, playlists, and user authentication.

## Features

- User authentication and authorization
- CRUD operations for songs, artists, and playlists
- Upload and manage MP3 files and images
- Integration with AWS S3 for file storage
- Token-based authentication

## Requirements

- Python 3.12.6
- Django 5.1.7
- Django REST Framework
- Pillow
- Boto3

## Installation

1. Clone the repository:

```sh
git clone https://github.com/nhattruong16062003/spotifyCloneBackend.git
cd spotifyCloneBackend
```
